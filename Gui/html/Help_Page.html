Help Page

Press any key to quit the help page.

The minimum amount required to plot any graph is to set the ydata parameter. This can be set the same as any other parameter, by typing the parameter followed by a colon, and the list of plotting parameters ended with a semi-colon. An example is given below:

parameter: item1, item2, ..., itemN ;

An example command to plot the Balance against the Date is given below:
ydata: Balance;
xdata: Date;
White space is ignored, so is the case of the characters (i.e. you can use capitalised or lowercase letters). Any line that begins with a '#' is ignored and can be used to comment.


The current list of commands are given as follows:

  • Ydata: Sets the data on the y-axis

  • Xdata: Sets the data on the x-axis

  • Act_Num: Chooses which account to plot data from


The data parameters can be set to any of the table headers on the table page. Alternatively set ydata or xdata to anything to get a full list of possible parameters.

To run plotting with the inputted parameters hit Cntrl-Enter or press the OK button. The backend package used to plot the data is matplotlib so I have kept the syntax pretty similar to that.
