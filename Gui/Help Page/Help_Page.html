Help Page

Press any key to quit the help page.

The minimum amount required to plot any graph is to set the ydata parameter. This can be set the same as any other parameter, by typing the parameter followed by a colon, and the list of plotting parameters ended with a semi-colon. An example is given below:

parameter: item1, item2, ..., itemN ;

An example command to plot the Balance against the Date is given below:
ydata: Balance;
xdata: Date;
White space is ignored, so is the case of the characters. Any line that begins with a '#' is ignored and can be used to comment.

The current list of parameters which can be set are given as follows:

  • Ydata -Sets the ydata to plot
  • Xdata -Sets the xdata to plot
  • Colour -Sets the colours of the plots
  • Linestyle (Ls) -Sets the linestyles of the plots.
The backend package used to plot the data is matplotlib so I have kept the syntax pretty similar to that.
